# Bayesian Neural Network with Variational Dropout
Implementing uncertainty estimation in deep learning using variational inference.

### 🔬 Research Highlights
- **Weight Distributions**: Replaces fixed weights with probability distributions to quantify model "epistemic" uncertainty.
- **Variational Dropout**: Models dropout as a Bayesian approximation to improve generalization and prevent overfitting.
- **Deep Learning Focus**: Critical for safety-critical AI where knowing "I don't know" is vital.

**Tech Stack:** Python, `PyTorch`, `TensorFlow Probability`.
